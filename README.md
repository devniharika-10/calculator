# 🧮 Calculator Web App

A simple and responsive calculator web application built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations with a clean and user-friendly interface.

---

## 🚀 Features

- Perform basic operations:
  - Addition (+)
  - Subtraction (−)
  - Multiplication (×)
  - Division (÷)
- Clear and delete functionality
- Responsive design for all screen sizes
- Smooth and interactive UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
